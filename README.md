# Desafio FastAPI DIO

Desenvolver uma API para o APP Workout utilizando FastAPI para dispor os dados de forma segura e ordenada.

Para este projeto foi utilizado as seguintes tecnologias:

- Python
- FastApi
- SQLAlchemy
- PostgreSQL
- Docker
- Git
- GitHub

Bibliotecas auxiliares:

- PyDantic
- Alembic
- Uvicorn
- Asyncpg
